Overview

Welcome to the Osiris project documentation. This project aims to provide a comprehensive platform for algorithmic trading strategy development, backtesting, and live execution with integrated AI/LLM capabilities for insights and decision support.

Components

The Osiris project consists of several key components:

  • LLM Sidecar: Provides AI-driven analysis, proposal generation, and text-to-speech services.
  • Osiris Policy Orchestrator: Manages the workflow for generating and evaluating trading proposals based on market data.
  • Simulation Engine: Simulates market data feeds for testing and development.
  • Web Interface & Tools: Including an audio console for real-time TTS feedback.

Tools and Consoles

  • Live Audio Console - Listen to real-time audio feedback from the system.
